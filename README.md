# THE RE-FORMER PROJECT

## Getting started

- Clone the repo `https://github.com/ZawadiSumba66/re-former.git`
- `cd` into the re-former
- Run `git checkout -b re-former`
- Run `git pull origin re-former`
- Run `bundle install`
- Run `rails db:migrate` to get the schema


## Features
- Start the rails server by running
```rails s```

### Create a new record
- To create a new record, go to http://localhost:3000/users/new
- A user can be able to enter there username,email and password and submit it.

### Edit an existing record
- To edit an existing record, go to http://localhost:3000/users/:id/edit
- A user can be able to edit there username,email or password and update it.
